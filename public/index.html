My Api Endpoints

  • Users
  • Articles
  • Topics
  • Comments

    • GET / api

    Returns a JSON object with a list of all available endpoints and their descriptions.

    GET / api / topics

    Returns an array of all topics with their descriptions.

    GET / api / articles

    Returns an array of articles with summary info: title, author, votes, etc.

    GET /api/articles/:article_id

    Returns detailed info about a specific article.

    PATCH /api/articles/:article_id

    Updates the votes of an article based on "inc_votes" in JSON body.

    {
        "inc_votes": number
      }

    GET /api/articles/:article_id/comments

    Lists all comments for an article, ordered by most recent.
